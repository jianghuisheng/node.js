### Promise

- 大量的异步操作 如果需要顺序执行 通过回调函数执行 会造成回调地狱
- 通过 Promise 解决回调地狱

1. 封装 Promise 函数

```
function test(){
  // 返回Promise
  return new Promise((resolve,reject)=>{
    // 需要的异步处理
    成功的时候 resolve('xxx')
    失败的时候 reject('xxx')
  })
}
```

2. 根据顺序 形成链式调用

```
   test().then().then().catch()
```

3. 捕获错误
